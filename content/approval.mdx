# **Stakeholder Map Approval Process**

## **Before submitting your stakeholder map for approval, make sure you’ve:**

- Selected and named all relevant stakeholder groups for each category
- Completed all required questions for each stakeholder (Tasks 2–7)
- Reviewed and edited responses for accuracy and clarity
- Flagged any entries that need further discussion or internal input
- Provided a rationale or description wherever prompted (especially if “Other” is selected)


## **Preparing for Submission**

<Step number="1" title="Complete All Required Sections">
Once everything is complete, click Submit for Approval. Users with Reviewer (Client) or Account Manager (CG) permissions will be notified and can approve or request changes.
</Step>
